# AI 提取策略（概要）

1) 目标：用 AI 自动理解页面语义并把活动信息映射成标准 JSON，而不是写死选择器。

2) 组合方法：
- 启发式解析：DOM 分块、候选容器打分、关键字段正则（日期/价格/地点/报名链接）。
- LLM 辅助抽取：把候选块与少量上下文发给 LLM，请求输出结构化 JSON（使用约束模式或 JSON Schema 校验）。
- 多语言：检测语言后在 LLM prompt 中指定目标语言与输出字段。
- OCR & PDF：对 PDF/图片使用 OCR（Tesseract 或商业 OCR），再走相同抽取流程。

3) Prompt 工程：给 LLM 明确 JSON schema，举 2-3 个示例输入输出，限制字段长度和格式。

4) 置信度与回退：LLM 应返回字段置信度、证据片段（HTML 片段或 DOM 路径）。低置信度转为人工审核队列或使用更保守启发式提取。

5) 可配置化：LLM 调用与策略应可通过 `agents/llm_client.py` 替换，支持本地或云端模型。
